What is CFP Scanner?

CFP Scanner is a community-powered database for storing information about upcoming conferences. A Call for Paper can be contributed by sending it via email to cfpscanner@mail.com. Once contributed, any relevant details will be extracted and then the event will be seen on the homepage.

How does it work?

CFP Scanner uses natural language programming techniques to extract information from a Call for Papers message. Each email is first checked using a pre-trained classifier to determine whether it contains information about a conference. If it does, a rule-based system is used to extract details and populate a database.
